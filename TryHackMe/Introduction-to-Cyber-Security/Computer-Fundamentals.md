# Computer Fundamentals

## Overview

This module introduced the basic components of a computer system and explained how computers communicate in different environments.

---

## Topics Covered

### ✅ Inside a Computer System

**Key Concepts**
- CPU (Central Processing Unit) executes instructions and processes data.
- RAM (Random Access Memory) temporarily stores data for running programs.
- Storage devices (HDD/SSD) permanently store the operating system, applications, and files.
- The motherboard connects all hardware components.
- Input devices allow users to provide data, while output devices display results.

---

### ✅ Computer Types

**Key Concepts**
- Desktop computers
- Laptops
- Servers
- Mobile devices
- Embedded systems

Each type of computer is designed for different purposes depending on performance, portability, and workload.

---

### ✅ Client-Server Basics

**Key Concepts**
- A **client** requests services or data.
- A **server** provides resources or services to clients.
- Communication occurs over a network.
- Common examples include:
  - Web browsers communicating with web servers
  - Email clients connecting to mail servers
  - Mobile apps connecting to backend servers

---

## Skills Gained

- Understanding of basic computer hardware
- Ability to distinguish different computer types
- Basic knowledge of client-server architecture
- Foundation for future networking and cybersecurity concepts
